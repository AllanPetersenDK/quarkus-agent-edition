# Product Phase 2

The old phase-2 product surface has been superseded by the canonical product lane in [`docs/product-lane.md`](docs/product-lane.md).

This file is kept as a historical note for the earlier phase-2 framing:

- Product conversation state is JDBC-backed.
- The schema is intentionally small and PostgreSQL-compatible.
- Local runtime and chapter smokes still run on the embedded H2 database.
- The operator lane remains read-only.
- Auth, roles, OIDC, and tenancy are intentionally phase-3 work.
